# Lab: Open Source Datasets & Deep Learning (IBM)

Este projeto faz parte da minha jornada na Certificação Profissional de Ciência de Dados da IBM. Explorei como consumir dados abertos e utilizar modelos de IA pré-treinados.

## 📊 Exercício 1: Exploração de Datasets
Trabalhei com um dataset público do aeroporto JFK (NOAA Weather Data) contendo **114.546 registros horários**.
- **Ferramenta:** Python / Pandas.
- **Foco:** Limpeza de dados e análise exploratória para previsão de atrasos em voos.

## 🤖 Exercício 2: Modelos de Deep Learning (MAX)
Utilizei o repositório **Model Asset Exchange (MAX)** para testar o modelo **Image Segmenter**.
- **Tecnologia:** Deep Convolutional Net treinada no dataset COCO.
- **Diferencial:** Diferente da detecção simples, a segmentação pixel-a-pixel separa o objeto (ex: pessoa, carro, cachorro) do plano de fundo.
